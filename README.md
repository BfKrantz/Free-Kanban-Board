# Kanban Board (Electron + React)

A desktop Kanban board app built with React and Electron. Made because I didn't want to pay for one.

## Features
- Multiple boards
- Drag-and-drop tasks
- Priorities & due dates
- Dark mode
- Import / export boards

## Development
- npm install
- npm start

## Build (Electron)
- npm run build
- npm run electron-build


## Installation Location
***Default install path:***
C:\Users\AppData\Local\Programs\kanban-app\

A Start Menu shortcut is created automatically.

To locate the installed files:
1.  Open Start Menu → search for **Kanban Board** 
2. Right‑click → **Open file location**

## Uninstall
The Board can be uninstalled via:
* Settings → Apps → Installed apps → Kanban Board
* Or Control Panel → Programs and Features
* Or using the uninstaller in C:\Users\AppData\Local\Programs\kanban-app\
